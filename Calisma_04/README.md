# Çalışma 4

## Soru

4 bitlik tek ve asal sayıların (1,3,5,7,11,13 sayılarının) çıkışlarını sıfır, tek basamaklı çift sayıların (0,2,4,6,8 sayılarının) çıkışlarını bir, diğer sayıların çıkışları için ise belirsiz değer veren devrenin tasarımını sadece nand kapıları ile tasarlayınız.

İsterler;

- Doğruluk tablosunda gösterim yapılması,

- Karnaugh map üzerinde sadeleştirme yapılması,

- Sadeleştirmeden sonra ifadenin sadece nand kapıları ile düzenlenmesi,

- Nand kapıları ile ifadenin modülünün kodlanması.
