# Sayısal Tasarım Uygulamaları

Bu repo, sayısal tasarım konularına yönelik Verilog tabanlı çalışmaların bir koleksiyonudur. Temel sayısal tasarım kavramlarının Verilog dili kullanılarak uygulanması amaçlanmıştır.

## İçerik

Her klasör bir çalışmayı temsil eder ve şu dosyaları içerir:
- `README.md` → İlgili sorunun metni ve görseller
- `.v` dosyaları → Verilog ile gerçekleştirilen çözüm
