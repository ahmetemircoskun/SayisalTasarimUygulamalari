# Çalışma 7

## Soru
(a) T Flip Flop modülü kodlayınız.

Girişler:
- T, fonksiyon girişi
- clk, clock pulse girişi
- reset, reset girişi

Çıkışlar:
- Q
- Qnot

(b) Yazdığınız modülü kullanarak 3 bitlik senkron yukarı sayıcı tasarlayınız. Sayıcınız 0’dan başlayarak 9 hane saymalıdır. Sayıcı bu değere ulaşınca reset ile başa dönmelidir.
