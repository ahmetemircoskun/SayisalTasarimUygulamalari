# Çalışma 8

## Soru
(a) JK Flip Flop modülü kodlayınız.

Girişler:
- J, fonksiyon girişi
- K, fonksiyon girişi
- clk, clock pulse girişi
- reset, reset girişi

Çıkışlar:
- Q
- Qnot

(b) Kodladığınız JK Flip Flop’u kullanarak 14-12-10-8-6-4-2-3-5-7-11-13-14-12… döngüsüne sahip 4 bitlik asenkron sayıcıyı tasarlayınız.

Sayaç girişleri:
- input clk, // Saat sinyali
- input rst, // Reset
- output [3:0] q // Sayaç çıkışı
